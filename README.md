1. Project Title

Hospitality Revenue & Performance Analytics Dashboard ‚Äì Multi-City Hotel Chain (2022)

2. Description

This Hospitality Analytics Dashboard provides a comprehensive, end-to-end analysis of hotel performance across multiple cities and properties.
The dashboard focuses on revenue, occupancy, pricing efficiency, room utilization, cancellations, and customer ratings, enabling hotel management to monitor operational health, identify underperforming properties, and optimize pricing and inventory strategies.
The project simulates a real-world hospitality analytics use case where business decisions are driven through data cleaning, modeling, KPI design, and interactive visual storytelling.

4. Data Source
Custom Synthetic Hospitality Dataset (2025) containing:
Property-level hotel data
City and room category details (Business / Luxury)
Revenue & pricing metrics
Occupancy and utilization metrics
Booking and cancellation indicators
Customer rating information
This dataset was designed to closely resemble real hotel management system (HMS) data.
